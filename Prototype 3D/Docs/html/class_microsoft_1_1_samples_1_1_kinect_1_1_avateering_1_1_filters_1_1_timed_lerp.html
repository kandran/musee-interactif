Detailed Description

TimedLerp - Maintains a time-based lerp between 0 and a upper limit between 0 and 1. The lerp speed parameter is in units of inverse time - therefore, a speed of 2.0 means that the lerp completes a full transition (0 to 1) in 0.5 seconds.
