Detailed Description

Filter to correct the skeleton position for the sensor height above the floor, to enable an avatar to be placed with feet on the ground plane.

CorrectSkeletonOffsetFromFloor moves the skeleton to the floor. If no floor found in Skeletal Tracking, we can try and use the foot position but this can be very noisy, which causes the skeleton to bounce up and down. Note: Using the foot positions will reduce the visual effect of jumping when an avateer jumps, as we perform a running average.

Parameters
skeletonThe skeleton to correct.
floorPlaneThe floor plane (consisting of up normal and sensor height) detected by skeleton tracking (if any).
avatarHipCenterHeightThe height of the avatar Hip Center joint.
