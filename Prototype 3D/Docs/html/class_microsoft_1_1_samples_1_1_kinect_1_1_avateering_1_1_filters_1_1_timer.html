Detailed Description

Class Timer is a helper class to perform timer operations For stop-watch timer functionality, use: Start() - To start the timer Stop() - To stop (or pause) the timer Reset() - To reset the timer Time - Returns current time or last stopped time For app-timing and per-frame updates, use: AbsoluteTime - To get the absolute system time GetAppTime() - To get the running time since construction (which is usually the start of the app) GetElapsedTime() - To get the time that elapsed since the previous call GetElapsedTime() call SingleStep() - To advance the timer by a time delta
